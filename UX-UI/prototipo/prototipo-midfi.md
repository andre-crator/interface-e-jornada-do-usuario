# Protótipo de Média Fidelidade

## Objetivo
Evoluir o protótipo de baixa fidelidade incorporando estrutura visual,
hierarquia de informação e melhor organização dos elementos.

---

## Características
- Média fidelidade
- Estrutura mais próxima do produto final
- Hierarquia visual definida
- Ainda sem identidade visual definitiva

---

## Telas Definidas

### 1. Onboarding
- Título claro
- Breve descrição do sistema
- Botão de ação principal destacado

---

### 2. Login
- Campos bem espaçados
- Ação principal em destaque
- Elementos secundários com menor peso visual

---

### 3. Dashboard
- Progresso exibido visualmente
- Matérias organizadas por prioridade
- Acesso rápido às ações principais

---

### 4. Gestão de Matérias
- Lista organizada
- Ações visíveis e padronizadas
- Feedback visual para ações do usuário

---

## Considerações
Este protótipo serve como base para:
- Ajustes visuais
- Testes adicionais
- Evolução para alta fidelidade

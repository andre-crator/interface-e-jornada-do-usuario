# Protótipo de Alta Fidelidade (High-Fidelity)

## Objetivo
Definir o protótipo visual detalhado do sistema, simulando a experiência final
do usuário com identidade visual, layout e interações.

---

## Tipo de Protótipo
- Alta fidelidade (High Fidelity)
- Layout próximo do produto final
- Uso de cores, tipografia e espaçamentos
- Interações visuais simuladas

---

## Diretrizes Visuais
- Layout limpo e responsivo
- Tipografia legível
- Ícones para facilitar a navegação
- Feedback visual em ações do usuário

---

## Telas Prototipadas

### 1. Login
- Campos com validação visual
- Mensagens de erro e sucesso
- Botão de destaque para ação principal

---

### 2. Dashboard
- Cards de resumo
- Gráficos simples de progresso
- Destaque para ações principais

---

### 3. Lista de Matérias
- Lista organizada por status
- Cores para diferenciar progresso
- Botões de ação rápida

---

### 4. Cadastro e Edição de Matéria
- Formulário organizado
- Feedback visual ao salvar
- Possibilidade de edição

---

### 5. Progresso da Matéria
- Barra de progresso visual
- Campo de observações
- Confirmação ao atualizar dados

---

## Interações Simuladas
- Hover em botões
- Estados ativos e desabilitados
- Mensagens de confirmação

---

## Observações
Este protótipo será a base para handoff com desenvolvimento
(front-end e back-end), servindo como referência visual e funcional.
